# Akari Ghostty Themes

Ghostty terminal emulator themes inspired by Japanese alleys lit by round lanterns.

## Installation

Copy the theme file to your Ghostty themes directory:

```bash
cp ghostty/akari-night ~/.config/ghostty/themes/
cp ghostty/akari-dawn ~/.config/ghostty/themes/
```

Then add to your Ghostty config:

```
theme = akari-night
```

Or for dawn:

```
theme = akari-dawn
```

## Variants

- **akari-night** - Dark theme with lantern-lit atmosphere
- **akari-dawn** - Light theme with morning warmth
