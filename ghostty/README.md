# Akari Ghostty Theme

> [!IMPORTANT]
> This repository is a read-only mirror.
> Issues, pull requests, and stars should go to [cappyzawa/akari-theme](https://github.com/cappyzawa/akari-theme).

Ghostty terminal emulator themes inspired by Japanese alleys lit by round lanterns.

## Installation

Clone the repository and symlink or copy the theme files:

```bash
git clone https://github.com/cappyzawa/akari-ghostty.git
cp akari-ghostty/akari-night akari-ghostty/akari-dawn ~/.config/ghostty/themes/
```

Then add to your Ghostty config:

```
theme = akari-night
```

Or for dawn:

```
theme = akari-dawn
```

## Variants

- **akari-night** - Dark theme with lantern-lit atmosphere
- **akari-dawn** - Light theme with morning warmth
