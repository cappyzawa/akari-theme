# Akari macOS Terminal Themes

macOS Terminal.app themes inspired by Japanese alleys lit by round lanterns.

## Installation

1. Double-click `Akari-Night.terminal` (or `Akari-Dawn.terminal`) to import the profile
2. Open Terminal > Settings > Profiles
3. Select the Akari theme and click "Default"

## Variants

- **Akari-Night.terminal** - Dark theme with lantern-lit atmosphere
- **Akari-Dawn.terminal** - Light theme with morning warmth
