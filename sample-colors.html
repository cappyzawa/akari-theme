# Python example showing all colors

from typing import List, Dict, Optional
import json

class DataProcessor:
    VERSION = "1.0.0"
    MAX_ITEMS = 100
    
    def __init__(self, config: Dict):
        self.config = config
        self.items = []
    
    def process(self, data: List[str]) -> Optional[Dict]:
        # Process data and return result
        for item in data:
            print(f"Processing: {item}")
            self.items.append(item)
        
        return {
            "count": len(self.items),
            "status": "complete"
        }

Color Comparison - Potential Issues

Blue/Cyan Family (4 colors)

#5A6F82

#6F879A
Cyan
#6F8F8A
Bright Cyan
#8FB8B0

These four colors are very close in hue (blue-cyan range). In code, cyan (#6F8F8A) is used for properties, bright_cyan (#8FB8B0) for types, bright_blue (#6F879A) for namespaces. The similarity may reduce visual distinction.
