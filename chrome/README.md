# Akari Chrome Themes

Chrome browser themes inspired by Japanese alleys lit by round lanterns.

Two variants are available:

## Night Theme

Night alley with warm shadows and lantern light.

**Installation:**

1. Open `chrome://extensions/` in Chrome
2. Enable "Developer mode" in the top right
3. Click "Load unpacked"
4. Select the `chrome/night/` directory

**Key colors:** Dark backgrounds (#171B22) with lantern orange active tabs (#E26A3B)

## Notes

- Both themes feature the signature lantern color for active tab highlighting
- Due to Chrome manifest v3 limitations, customizable elements are limited
- For more detailed color control, consider using CSS extension tools like Stylus
- Only one theme can be active at a time
